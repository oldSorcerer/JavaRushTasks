
### Подготовка Кибердракона

Нужно написать программу отстрела десантных кораблей галабагов. У кораблей галабагов есть уязвимости для разных фаз плазменных зарядов Кибердракона.
Нужно проверять в какой фазе находится корабль галабага и соответствующе менять фазу плазмы которой стреляет Кибердракон.
Программируем Кибердракона чтобы он плевал разной плазмой по разным типам десанта галабагов.
Верное выполнение задачи приводит к тому, что Кибердракон стреляет правильной плазмой и уничтожает всех галабагов.
В этой задаче нужно правильно зарядить пушки (массивы-строки) зарядами (числа). Поехали...
Дан двумерный массив array (массив, который хранит ссылки на массивы-строки). Необходимо его проинициализировать массивами-строками,
длина которых равна 2 * &lt;номер массива-строки&gt; + 1. То есть array[0] будет иметь длину 2 * 0 + 1 = 1, array[1] будет иметь длину 2 * 1 + 1 = 3 и т.д.
Заполни все массивы-строки числами, которые равны номеру элемента в массиве-строке + номер самой строки.
Т.е. массив-строка под номером 3 будет заполнена вот так:
{3, 4, 5, ...}
(0+3, 1+3, 2+3, ...)


Требования:
1.	В классе Solution должна быть публичная статическая переменная int[][] array, проинициализированная значением int[5][].
2.	Проинициализируй массивы-строки, которые хранит переменная array, массивами необходимой длины (смотри условие).
3.	Заполни массивы-строки числами согласно условию.
