
### Оркестр с бубенцами

Разберись, что делает программа. Попробуй запустить.
Тебе нужно реализовать метод completeConcert, который должен проверить все задания в полученном списке.
Если задание еще не завершено, то completeConcert должен завершить его принудительно.


Требования:
1.	Метод completeConcert должен проверить каждую задачу полученного списка: завершена ли она.
2.	Метод completeConcert должен досрочно завершить все незавершенные задачи.
3.	Метод completeConcert не должен ничего делать с уже завершенными задачами.
